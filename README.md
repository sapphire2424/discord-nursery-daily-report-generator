# Discord Daily Report Generator

Discord上のやり取りを自動収集し、生成AI（GPT-4o）を用いて業務日報を自動生成するシステムです。

## 概要
保育現場などの忙しい環境において、スタッフ間のチャットログからその日の重要な出来事や連絡事項を抽出し、適切なフォーマットの日報として自動出力します。これにより、事務作業の負担を大幅に軽減することを目的としています。

## 主な機能
- **ログの自動収集**: 指定したサーバー内のテキストチャンネルおよびフォーラムチャンネル（アーカイブ済みを含む）からメッセージを取得。
- **AIによる要約**: OpenAI APIを活用し、膨大なログから「事実」に基づいたコンテキスト解析を行い、日報を生成。
- **自動送信**: 生成された日報を指定のDiscordチャンネルへ自動投稿。

## 技術スタック
- **言語**: Python
- **ライブラリ**: discord.py, OpenAI SDK
- **インフラ/外部ツール**: Discord API, OpenAI API (GPT-4o)

## 実装のこだわり
メッセージの取得漏れを防ぐため、スレッドやアーカイブされたフォーラムの取得ロジックを最適化しました。また、プロンプトエンジニアリングにより、AIのハルシネーション（嘘）を抑え、ログに存在する情報のみを抽出するように制約を設けています。
